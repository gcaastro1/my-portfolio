# Gustavo Lima — Portfolio

Personal portfolio for Gustavo Lima, a Front-End Developer focused on React.js,
Next.js and TypeScript. The website presents professional experience, technical
skills and selected projects with a responsive interface and light/dark themes.

## Tech stack

- Next.js
- React
- TypeScript
- Sass / SCSS Modules
- Framer Motion
- Phosphor Icons
- Vercel

## Features

- Professional introduction and downloadable CV links in Portuguese and English
- About and technical skills sections
- Curated featured projects
- Professional experience timeline
- Direct contact links for LinkedIn, email and WhatsApp
- Responsive layout with light and dark themes

## Running locally

Requirements: Node.js 18+ and npm.

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

To create a production build:

```bash
npm run build
npm start
```
